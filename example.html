This is a very simple example of MQTT. Click Connect, then use the Subscribe buttons. You should see text appear below.

Соединение с сервером:

- Нажмите кнопку "Connect".

- При успешном подключении к серверу в диалоговом окне ниже будет получен ответ "CONNACK 0".

- Измените первый раздел топика на свой Email и нажмите кнопку "Subscribe". В окне ниже будет виден MQTT трафик.

- При разрыве соединения или отключении будет получен ответ "Lost connection".

Получение данных, переданных по протоколу MQTT:

- Замените в форме, находящейся ниже, первый раздел топика на свой Email, зарегистрированный в FlyMon DMS.

- Не изменяйте остальные два раздела топика "/+/#", если хотите получать данные по всем устройствам.

- Замените раздел топика "+" на "DeviceName" (имя своего устройства), если хотите получать данные по конкретному устройству.

- При подписке, сразу будут отражены данные с устройства, полученные последними на сервере (за месяц).

Получение данных, переданных по протоколу FlyMon:

- Замените в форме, находящейся ниже, первый раздел топика на Email "demo@flymon.net".

- Замените второй раздел топика "+" на MAC вашего устройства, написанный в нижнем регистре (маленькими буквами), без двоеточий.

- При подписке, данные с устройства появятся только в момент их передачи (интервал 5 мин.).

Дополнительные пояснения для обоих протоколов:

- Третий раздел топика "/#" трогать не нужно, если хотите видеть все метрики с устройства.

- Если есть необходимость, можно заменить раздел топика "#" на название контретной метрики, сенсора или gpio.

- Все данные в разделах топика вводятся без кавычек "".

Отправка данных:

- Измените первый раздел топика на свой Email, укажите имя устройства, метрику и значение. Нажмите кнопку "Publish".

Фичи:

- если подписаться на топик "timestamp" (одно слово и без кавычек), то каждую минуту будут приходить временные метки.
