Функциональные возможности IoT хаба Araneus ("брикет")

 Организация канала связи tun/tap для удалённого администрирования сети клиента Передача скриншотов с UVC WEB-камер, подключенных к USB порту, на сервер FlyPeek Наличие необходимых драйверов и утилит для прошивки модулей ESP8266, подключаемых к USB Локальный MQTT сервер (все устройства соединяются только с локальным брокером) WiFi точка (доступа для модулей ESP8266) Канал связи 3G через GPRS/CDMA модемы, подключенные к USB порту Интерфейс I2C для подключения датчиков и исполнительных устройств
